# Fscan 参数对比分析

## ❌ 前端需要删除的参数（fscan 中不存在）

| 参数 | 前端显示名称 | 说明 |
|------|------------|------|
| `top` | 存活优先扫描数量 | fscan 中不存在此参数 |
| `ping` | 使用Ping | fscan 只有 `-np` 禁用ping，没有启用选项 |
| `fingerprint` | 启用指纹识别 | fscan 中已删除，服务指纹识别默认启用 |
| `pg` | 显示进度条 | fscan 只有 `-nopg` 禁用进度条 |
| `sp` | 显示扫描计划 | fscan 中不存在此参数 |
| `slow` | 慢速日志输出 | fscan 中不存在此参数 |

## ➕ 前端需要添加的参数（fscan 有但前端没有）

### 目标配置参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-ehf` | 排除主机文件 | "" |
| `-ep` | 排除端口 | "" |

### 扫描控制参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-ao` | 仅存活主机模式 | false |
| `-local` | 本地插件名称（需改为文本输入） | "" |

### 认证与凭据参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-upf` | 用户密码对文件 | "" |
| `-hashf` | hash文件 | "" |
| `-hash` | hash值 | "" |

### Web扫描参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-max-redirect` | 最大重定向次数 | 10 |

### 发包频率控制参数（新增分组）
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-rate` | 发包速率限制（每分钟最大发包数） | 0 |
| `-maxpkts` | 整个程序最大发包总数 | 0 |

### 输出与显示控制参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-nopg` | 禁用进度条（替代 pg） | false |

## 🔧 需要修正的默认值

| 参数 | 前端当前值 | fscan 实际值 | 需要修改 |
|------|----------|-------------|----------|
| `-t` | 10 | 600 | ✅ 是 |
| `-mt` | 10 | 20 | ✅ 是 |
| `-log` | 'SUCCESS' | 'LogLevelBaseInfoSuccess' | ✅ 需要检查 |

## 📋 高级功能参数（可选添加）

### 后渗透与高级功能
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `-sc` | Shellcode | "" |
| `-rsh` | 反弹Shell目标 | "" |
| `-start-socks5` | 启动SOCKS5服务器端口 | 0 |
| `-fsh-port` | 正向Shell端口 | 4444 |
| `-persistence-file` | 持久化文件 | "" |
| `-win-pe` | Windows PE文件 | "" |
| `-keylog-output` | 键盘记录输出 | "keylog.txt" |
| `-download-url` | 下载文件URL | "" |
| `-download-path` | 下载文件保存路径 | "" |
| `-lang` | 语言设置 | "zh" |

## 📝 修改优先级

### P0 - 必须修改（影响参数正确性）
1. 删除不存在的参数：top, ping, fingerprint, pg, sp, slow
2. 修正默认值：t (10→600), mt (10→20)

### P1 - 重要添加（常用功能）
1. 添加排除参数：ehf, ep
2. 添加认证参数：upf, hashf, hash
3. 添加 Web 参数：max-redirect
4. 添加扫描控制：ao, local（文本输入）
5. 添加输出控制：nopg（替代 pg）

### P2 - 可选添加（高级功能）
1. 发包控制：rate, maxpkts
2. 后渗透功能：sc, rsh, start-socks5, fsh-port, etc.
